# Support

VoxCPM Batch Cloning UI is open source and free to use.

If the project helps you, donations are welcome but never required:

- [Donate with PayPal](https://www.paypal.com/donate/?hosted_button_id=JZRCHDWDQKUGN)

You are also welcome to:

- fork the project,
- adapt it for your own workflow,
- open issues for bugs or documentation gaps,
- and submit improvements through pull requests.

If you publish a fork, please keep the license terms intact and only redistribute reference-voice recordings you have the right to share.
