# Voice Samples

This public project does not include voice recordings.

Place your own clean reference clips in this folder before using the local `Voice Sample` dropdown in the app.

Recommended for this project:

- at least `20 seconds` of clear speech,
- only one speaker,
- no music,
- little or no background noise,
- supported formats: `.wav`, `.mp3`, `.ogg`, `.m4a`.

For Hi-Fi mode, add an optional same-name transcript file if you want the interface to auto-fill the exact spoken words:

```text
my_voice.wav
my_voice.txt
```

Only use and share voice recordings you own or have permission to use.
